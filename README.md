# BMW Api npm package

## Description

A Typescript library that encapsulates the BMW RESTful API

Inspiration:
 1. [BMW-i-Remote](https://github.com/edent/BMW-i-Remote)
 2. [bimmer_connected](https://github.com/bimmerconnected/bimmer_connected)

## Installation


```bash
$ npm install wrapper-bmw
```

## Documentation

```bash
$ npm run doc
```
